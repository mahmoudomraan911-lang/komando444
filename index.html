قسم أدوات وبرامج Python و GitHub المخصصة

Evilginx2 Tool

أداة متقدمة جداً لاختبار اختراق الحسابات عن طريق محاكاة هجمات التصيد وتخطي ميزة التحقق الثنائي (2FA) عبر اعتراض الكوكيز والـ Tokens.

DarkFly Tool

أداة قوية جداً مجمعة ومخصصة لتثبيت واختيار ما يصل إلى 542 أداة اختراق وهندسة عكسية وتوعية أمنية مختلفة بنقرة واحدة.

GhostTrack (OSINT)

أداة استخباراتية شهيرة ومفتوحة المصدر لجمع معلومات دقيقة وتتبع المواقع وفحص البيانات الرقمية المتاحة علناً.

PhoneInfoga (Python)

أداة متطورة جداً لجمع المعلومات وفحص أرقام الهواتف وتحديد النطاق الجغرافي والبيانات المتاحة عنها مفتوحة المصدر.

Maltego API Tool (Python)

سكريبتات ربط مخصصة بلغة بايثون لتحليل البيانات الضخمة ورسم الخرائط البيانية لجمع المعلومات الاستخباراتية.

Nmap Port Scanner (Python)

أداة فحص منافذ الشبكة وسيرفرات الويب الشهيرة، مبرمجة ومدمجة بالكامل لتعمل داخل بيئة سكريبتات بايثون الذكية.

Scapy Packet Crafting (Python)

مكتبة وأداة بايثون جبارة لفحص الحزم البرمجية داخل الشبكة، وتعديلها واختبار جدران الحماية ضد الثغرات.

Check Links Tool

سكريبت بايثون ذكي مخصص لفحص الروابط المشبوهة وتحليلها للتأكد من سلامتها وخلوها من سكريبتات التتبع أو ملفات الضارة قبل فتحها.
